ToDoPlanner App

-Description
 --The ToDoPlanner app is a simple and intuitive task management application designed to help users organize their daily tasks and boost productivity.

-Features
 --Task Creation: Easily create new tasks with a title and description.
 --Task Management: Mark tasks as completed, delete unnecessary tasks, and edit task details.
 --Prioritization: Assign priority levels to tasks to focus on the most important ones.
 --Due Dates: Set due dates for tasks to stay on top of deadlines.
 --User-Friendly Interface: Intuitive design for a seamless user experience.

-Usage
 --Add a new task by clicking on the "Add Task" button.
 --Edit task details by clicking on the task.
 --Mark tasks as completed by checking the checkbox.
 --Delete tasks by clicking on the "Delete" button.
 --Prioritize tasks using the priority dropdown.
 --Set due dates for tasks by clicking on the calendar icon.
